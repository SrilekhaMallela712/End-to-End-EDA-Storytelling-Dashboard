# End-to-End-EDA-Storytelling-Dashboard
An end-to-end Exploratory Data Analysis (EDA) project using Python, Pandas, and visualization libraries to analyze a housing dataset. It includes data cleaning, statistical analysis, and interactive insights, culminating in a Streamlit dashboard for effective data storytelling and business understanding.
# 🏠 Housing Data Analysis Dashboard

## 📊 Project Overview

This project focuses on performing end-to-end Exploratory Data Analysis (EDA) on a housing dataset and building an interactive dashboard using Streamlit. It covers data cleaning, statistical analysis, visualization, and data storytelling.

---

## 🚀 Features

* 📄 Dataset overview and summary statistics
* 🔍 Interactive data filtering (price range & furnishing status)
* 📊 Dynamic visualizations (histogram, scatter plot, boxplot)
* 📥 Download filtered dataset as CSV
* 📌 Multi-page Streamlit dashboard

---

## 🛠️ Technologies Used

* Python
* Pandas
* Seaborn
* Matplotlib
* Streamlit
* Scipy

---

## 📁 Project Structure

* `app.py` → Streamlit dashboard
* `Housing.csv` → Dataset
* `cleaned_housing.csv` → Cleaned dataset
* `eda_dashboard_project.py` → EDA code (Day 1–4)

---

## ▶️ How to Run

1. Clone the repository
2. Install dependencies

   ```bash
   pip install pandas seaborn matplotlib streamlit scipy
   ```
3. Run the application

   ```bash
   streamlit run app.py
   ```

---

## 📈 Key Insights

* House prices are strongly influenced by area and number of rooms
* Furnishing status impacts price distribution
* Statistical tests show significant differences between categories
